lieweb
======

## Overview

[`lieweb`] is a simple async web framework in Rust-lang.

## Getting Started

Examples can be found in [`examples`].

## Rust Version

`lieweb` currently works on rust `1.39` and above as it requires support for the `async_await`
feature.
